In order to distribute the work effectively, we organize the project into sprints.
Each sprint defines a set of tasks to be completed during a specific week of the project.
Within every sprint, we establish clear objectives that should be achieved by the end of the
week and track the progress made throughout that period.

<div class="callout tip">
	<div class="callout-title">Note</div>
	Although the sprints are defined in this document, we use Jira to better track
	task management, including who is responsible for each task and the progress being made.
	Therefore, this section should be considered primarily as a record of the project's
	development history.
</div>
